About

This website hosts a bunch of watercolor style stains used for Dungeons & Dragons homebrew material. It is specifically designed for use with GM Binder and Homebrewery. Here is the full guide I wrote up on how to use them.

Originally I had put 42 files in this imgur album (they are still there!). Unfortunately, GM Binder is not able to directly link to imgur. So when I finally got around to making some more content, I decided I'd try to throw this together. It might not work forever, but it is hosted by GitHub Pages for now.
